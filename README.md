## 🏗️ Getting Started

To run the portfolio locally:

```bash
git clone https://github.com/praveen-1826/Praveen-s-Portfolio.git
cd Praveen-s-Portfolio
npm install
npm start
```

Visit `http://localhost:3000` in your browser to view the site.
